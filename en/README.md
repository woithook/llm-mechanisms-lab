# LLM Notebook

Personal notes documenting my exploration, implementation, and research on Large Language Models (LLMs).

## Overview

This repository contains a collection of IPython Notebook (`.ipynb`) memos related to large language models and modern AI systems.

The notebooks in this repository are translated from my original Chinese memo-style notebooks with the assistance of large language models (LLMs). They are mainly used to document my learning process, technical experiments, paper reading notes, implementation details, and research-oriented reflections.

## Topics

The repository may include notes on topics such as:

* Large language models
* Tokenization and text compression
* BPE and subword modeling
* RoPE and positional encoding
* Transformer implementation details
* Multimodal learning
* Vision-Language-Action models
* Reinforcement learning
* AI agent systems
* Research notes and technical tutorials

## Purpose

The goal of this repository is to build a personal knowledge base for understanding LLMs from both theoretical and implementation perspectives.

By translating my original Chinese notebooks into English, this repository also makes the notes easier to share, review, and reuse in a broader research and engineering context.

## Notes

Since the notebooks were translated with the help of LLM-based tools, some wording, technical terminology, or code comments may still require manual review. The contents should be understood as personal study memos rather than polished tutorials or formal publications.

## Disclaimer

This repository is mainly for personal learning and research documentation. The contents may contain incomplete explanations, experimental code, translation inaccuracies, or non-standard terminology. Please verify important technical details before relying on them for research, teaching, or engineering work.
